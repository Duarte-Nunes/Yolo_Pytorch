# Instalation
1. Dowload the repository

    `https://github.com/Duarte-Nunes/Yolo_Pytorch.git`
    
2. Install the Requirements 

    `cd Yolo_Pytorch`
    
    `pip install -r requirements.txt`
    
4. Make sure Pytorch was compiled with CUDA enabled
